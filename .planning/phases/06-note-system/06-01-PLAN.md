2. Add an `input` event listener to `#card-note` that updates the object and saves to `localStorage`.
3. Update `openCard(card)` to populate the textarea with the correct note for that card index.

**Logic snippet:**
```javascript
const noteTextarea = document.getElementById('card-note');
let cardNotes = JSON.parse(localStorage.getItem('investigation_notes') || '{}');

noteTextarea.addEventListener('input', () => {
  if (currentCardIndex === null) return;
  cardNotes[currentCardIndex] = noteTextarea.value;
  localStorage.setItem('investigation_notes', JSON.stringify(cardNotes));
});

// Inside openCard:
noteTextarea.value = cardNotes[currentCardIndex] || '';
```
